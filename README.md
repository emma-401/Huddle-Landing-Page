#frontendmentor.io

This is a challenge from Front-end Mentor which involved me replicating a design to look like the original image. 

The design is replicating a Huddle Landing Page with Single Introductory Section. It's original design is found in the image folder. The goal of this design is to practise FLexbos by creating responsive layout.

This design challenged me because it made me learn new things using HTML and CSS. Like the use of target="_blank" in href for creating of link, this enables the link to open up in another browser inside of opening in that same broswer and taking the user completely out of the page. 

Also, the ways used to section each work apporpiately in html using div which enabled me in stylying them distinctly and as a group in CSS.
